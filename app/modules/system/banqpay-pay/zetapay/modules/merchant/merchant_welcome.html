Remember Search Criteria

You don't have to re-enter the same frequently used parameters each time you run an Activity Report. You can now save your search criteria by selecting the "Remember my search criteria" check box. This feature remembers all the search parameters in the report, so you can, for example, set a merchant account, date range, and card type as your default search criteria. These criteria are saved for the duration of your current session in the back office and are automatically set when you return to the Activity Report.
Time Zone

By default, the Date & Time parameters for the Activity Report are set to Eastern Time (Montreal and New York, e.g., are in this time zone). However, you can now adjust this setting so that if you are in a different time zone, you don't have to calculate the range from a report run with Eastern Time settings - you just run the report with your own time zone settings selected, and the report results will reflect this.
First Name, Last Name Search

In order to make your Activity Report searches more effective, we've split the Name field into two: First Name and Last Name. This allows you to provide even more detail when you are searching for a particular transaction for a particular customer, which means faster reporting times and more accurate results.
