User Features Description

 Projects

Extensive project features implemented to make your project management work easier than before. Features include:

  • Unlimited Task Hierarchies
  • Project Templates
  • Consolidated and Non-Consolidated Hours Calculation
  • User details on estimated and actual hours calculations per user
  • Email Alerts
  • Logging Capabilities

 Timesheet

Timesheet management system at its best. Features include:

  • Single Dashboard View for All Timesheet Status
  • Client Verification for Timesheet Entries
  • Admin Control for User's Timesheet Hours Entry
  • Holiday Management
  • Importing Previous Week's Timesheet Data
  • Weekly or Daily Timesheet Views
  • Email Alerts
  • Logging Capabilities

 Approvals

Approvals for timesheet. Features include:

  • Projects Based Approvals
  • Supervisor Based Approvals
  • System (Automated) Based Approvals
  • Email Alerts
  • Logging Capabilities

 Reports

Extensive report features with user requested reports. Features include:

  • Project Reports
  • Timesheet Reports
  • Log Reports

 Preference

User preference to suit different users' preferences. Features include:

  • Date Format
  • Name Format
  • Currency Settings
  • Weekly or Daily Timesheet Views
  • Decimal and Thousand Separator
