1. General Preference

     
	Figure 1: General User Preference Page
  1. Date Format: Date display preference. All dates in system will follow this format
  2. Name Format: Name display preference. All names in system will follow this format
  3. Currency Preference: Default currency. Any currency selection will default to this selected currency symbol
  4. List Max Lines Per Page: All listings in page will have the total list as shown. Listings that exceed the number will overflow to the next page
  5. Default Thousand Separator (SPC For Space): Default separator for thousands
  6. Default Decimal Separator (SPC For Space): Default separator for decimal numbers
  7. Task Wrapping: Tasks will wrap automatically to a new line during Timesheet activities
  8. Show Only Open Tasks In Open Projects List?: Only Tasks with Open status will be shown during Project Selections
  9. Expand All Project Tasks By Default?: Project Treeview to expand or collapse by default
