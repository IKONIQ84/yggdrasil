Scrap WebSite Analyzer

    Information

    This script scrape a website and return for each page:

    • the title
    • the description
    • h1, h2 and h3 tag
    • header status

    And the list of image with alt tag, script and style wth their size.

    You can also export as CSV each table

    Configure

    Create a new database, for example with phpmyadmin.

    Open config.php and insert the host name (eg: localhost).

    The username, the password and the databse name ($db_name).

    Usage

    Go to the url of the script and insert:

    The url (eg: http://www.example.com). Remember to insert http://www. and not use the last /.

    Choose the recursion level. If it is set to 0 only scrap the home page link. If it is set to 1 scrap the home page link and all the link of the linked page.

    With the Separator you can extract a portion of the text, such an article. You only need to insert the class you wants to get.
    For example: i wants to get the article from http://net.tutsplus.com. So i check the code for find the article separator, i see that all the article is inside "single_post" class. So in Separator i insert ".single_post".

    Prefix is for the www or not www content. In the case the site doesn't use www. i leave it blank (for example for net.tutsplus.com) else i insert "www.".

    Example: extract all the information and the post from net.tutsplus.com:
    Url: http://net.tutsplus.com
    separator: .single_post
    Prefix: leave it blank

    Now run, wait and enjoy!

     

    Limit

    No work if the url use # character.
